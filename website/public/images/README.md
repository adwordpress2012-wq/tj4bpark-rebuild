# TJ4B Park Image Placeholders

Use these folders for real client-supplied TJ4B Park photography when the site moves from mockup to review-ready production content.

- `hero/` - primary first-screen park image. Recommended file: `tj4b-park-hero.jpg`, landscape, at least 2400px wide.
- `activities/` - activity cards for camping, Gumnut's glamping, 4WD tracks, buggy rides, bushwalks, wildlife, and group events.
- `gallery/` - candid park moments for the gallery mosaic, mixing wide, square, and portrait crops.
- `branding/` - logo marks, Open Graph/social preview image, favicon source files, and any approved brand assets.

Keep filenames lowercase and descriptive, for example `camping-howes-valley.jpg` or `buggy-rides-tj4b-park.jpg`. Add image paths to `lib/tj4b-park.ts` once the approved photos are available.
